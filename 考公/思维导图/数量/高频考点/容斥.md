# 特征：集合间有交叉
# 解题：公式、画图
# 题型
## 二者容斥
### 两个集合之间有交叉重叠
### 公式
#### A+B - A交B + 都不 = 全集
#### 只A + 只B + A交B + 都不 = 全集

## 三者容斥
### 三个集合之间有交叉重叠
### 公式
#### A + B + C - A交B - A交C - B交C + A交B交C + 都不 = 全集
##### 触发条件：题目分开给出 两两相交部分的值
#### A + B + C - 只参与两项的值 - 2 A交B交C + 都不 = 全部
##### 触发条件：题目一起给出 只参与两项的部分的值
