### 三集合的容斥公式是什么
大：A|B|C = A + B + C - A&B - B&C - A&C + A&B&C
小：A|B|C = A + B + C - (A&B-A&B&C) - (B&C-A&B&C) - (A&C-A&B&C) - 2 * A&B&C
