# 信的主要写作目的
## 向读者宣传、推介某件事，类似宣传稿
## 针对读者的问题、质疑或社会舆论做出回应
# 标题：给/致+对象+的+文种
# 开头：背景+反应的问题以及对待问题的态度+引导句
## 交代反应的问题以及对待问题态度=反映的XX问题，XX高度重视当即展开调查，现将......回应如下：
# 主体：解释初夏那问题的原因、理由+解决问题的对策（双要素，分段呈现）
# 结尾：总结+感谢监督/欢迎监督
