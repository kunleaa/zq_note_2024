# 识别：时间在材料之前+“倍”
# 列式：基期A/基期B
# 题型1 已知现期A、B，以及对应的增长率a，b
## 基期A是基期B的几倍=A/B x (1+b)/(1+a)
## 基期A比基期B多几倍=A/B x (1+b)/(1+a) -1
# 题型2 已知现期A、B，以及对应的增长量$\Delta A，\Delta B$
## 基期A是基期B的几倍=$\frac{A-\Delta A}{B-\Delta B}$
## 基期A比基期B多几倍=$\frac{A-\Delta A}{B-\Delta B}-1$
